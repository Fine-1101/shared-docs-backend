# 前端修改文档：手动保存版本功能

## 📋 修改概述

将历史版本保存策略从**定时自动保存**改为**手动触发**，用户通过点击保存按钮或退出文档时才会创建历史版本。

---

## 🎯 修改目标

1. ✅ 添加"保存版本"按钮
2. ✅ 实现手动保存版本功能
3. ✅ 用户退出时自动保存版本
4. ✅ 显示保存结果提示

---

### 3. 实现保存版本逻辑

**文件位置**：`src/views/DocumentEditor.vue` 的 `<script>` 部分

```typescript
<script setup lang="ts">
import { ref, onMounted, onBeforeUnmount } from 'vue';
import { ElMessage } from 'element-plus';  // 或其他 UI 库的消息提示

// 状态定义
const isSaving = ref(false);
const currentVersion = ref<number>(0);

// ✅ 保存版本功能
const handleSaveVersion = () => {
  if (isSaving.value) {
    ElMessage.warning('正在保存中，请稍候...');
    return;
  }
  
  isSaving.value = true;
  
  try {
    // 通过 WebSocket 发送保存版本请求
    const message = {
      type: 'save_version',           // 注意：小写下划线
      documentId: props.docId,
      userId: authStore.userId,
      timestamp: Date.now()
    };
    
    ws.send(JSON.stringify(message));
    
    console.log('📤 已发送保存版本请求');
    
    // 超时保护：如果 5 秒内没有收到响应，重置按钮状态
    setTimeout(() => {
      if (isSaving.value) {
        ElMessage.warning('保存超时，请检查网络连接');
        isSaving.value = false;
      }
    }, 5000);
    
  } catch (error) {
    console.error('❌ 发送保存版本请求失败:', error);
    ElMessage.error('保存版本失败，请重试');
    isSaving.value = false;
  }
};

// ✅ 监听 WebSocket 消息
const handleWebSocketMessage = (event: MessageEvent) => {
  try {
    const message = JSON.parse(event.data);
    
    switch (message.type) {
      case 'system_notification':
        // 处理系统通知（保存结果）
        handleSystemNotification(message);
        break;
        
      case 'error':
        // 处理错误消息
        ElMessage.error(message.data?.message || '操作失败');
        isSaving.value = false;
        break;
        
      // 其他消息类型处理...
      default:
        break;
    }
  } catch (error) {
    console.error(' 解析 WebSocket 消息失败:', error);
  }
};

// ✅ 处理系统通知（版本保存结果）
const handleSystemNotification = (message: any) => {
  const notification = message.data;
  
  // 显示保存结果提示
  if (notification.includes('已保存')) {
    ElMessage.success(notification);
    // 更新当前版本号（从通知中提取）
    const match = notification.match(/版本 (\d+) 已保存/);
    if (match) {
      currentVersion.value = parseInt(match[1]);
    }
  } else if (notification.includes('未变化')) {
    ElMessage.info(notification);
  } else {
    ElMessage.info(notification);
  }
  
  // 重置保存按钮状态
  isSaving.value = false;
};

// ✅ 用户离开文档时自动保存版本
const leaveDocument = () => {
  if (ws.readyState === WebSocket.OPEN) {
    ws.send(JSON.stringify({
      type: 'leave_document',
      documentId: props.docId,
      userId: authStore.userId,
      timestamp: Date.now()
    }));
    console.log(' 已发送离开文档请求（将自动保存版本）');
  }
};

// ✅ 页面卸载时确保发送离开消息
onBeforeUnmount(() => {
  leaveDocument();
  if (ws) {
    ws.close();
  }
});

// ✅ 监听浏览器关闭事件
window.addEventListener('beforeunload', () => {
  if (ws.readyState === WebSocket.OPEN) {
    ws.send(JSON.stringify({
      type: 'leave_document',
      documentId: props.docId,
      userId: authStore.userId,
      timestamp: Date.now()
    }));
  }
});

// ✅ 组件挂载时初始化
onMounted(() => {
  // 设置 WebSocket 消息监听
  if (ws) {
    ws.onmessage = handleWebSocketMessage;
  }
  
  // 加载当前版本信息（可选）
  loadCurrentVersion();
});

// 加载当前版本（可选）
const loadCurrentVersion = async () => {
  try {
    const response = await axios.get(`/api/versions`, {
      params: {
        documentId: props.docId,
        page: 1,
        pageSize: 1
      }
    });
    
    if (response.data.data.list.length > 0) {
      currentVersion.value = response.data.data.list[0].versionNumber;
    }
  } catch (error) {
    console.error('加载版本信息失败:', error);
  }
};
</script>
```

## 🔌 WebSocket 消息格式

### 发送消息（前端 → 后端）

**保存版本请求**：
```json
{
  "type": "save_version",
  "documentId": 123,
  "userId": 456,
  "timestamp": 1715000000000
}
```

**离开文档请求**：
```json
{
  "type": "leave_document",
  "documentId": 123,
  "userId": 456,
  "timestamp": 1715000000000
}
```

### 接收消息（后端 → 前端）

**保存成功**：
```json
{
  "type": "system_notification",
  "messageId": "uuid-xxx",
  "documentId": "123",
  "userId": 456,
  "timestamp": 1715000000000,
  "data": "版本 5 已保存"
}
```

**内容未变化**：
```json
{
  "type": "system_notification",
  "messageId": "uuid-xxx",
  "documentId": "123",
  "userId": 456,
  "timestamp": 1715000000000,
  "data": "内容未变化，未创建新版本"
}
```

**错误消息**：
```json
{
  "type": "error",
  "messageId": "uuid-xxx",
  "documentId": "123",
  "userId": 456,
  "timestamp": 1715000000000,
  "data": {
    "code": 500,
    "message": "保存版本失败: 文档不存在"
  }
}
```

## ⚠️ 注意事项

1. **消息类型格式**：必须使用小写下划线 `save_version`，不能使用 `SAVE_VERSION`
2. **不需要传递内容**：后端会自动从数据库读取当前文档内容
3. **按钮禁用逻辑**：保存过程中禁用按钮，防止重复提交
4. **超时保护**：如果 5 秒内没有收到响应，提示用户
5. **退出时自动保存**：确保所有退出场景都发送 `leave_document` 消息
6. **用户体验**：保存成功后给用户明确的反馈
